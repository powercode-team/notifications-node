## Development

### Prepare with Lerna

- `yarn`
- `lerna run link`
- `lerna link`
- `lerna bootstrap`

### Build for Development

- `lerna run build-dev`
- `lerna run rebuild-dev`

### Build for Production

- `lerna run build`
- `lerna run rebuild`

### Lerna Publish

- `lerna run rebuild && lerna publish --force-publish`
